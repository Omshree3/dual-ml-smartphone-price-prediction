# Dual ML System for Smartphone Price Prediction

This project uses both Regression and Classification models to predict smartphone price and price range based on various features.

## ML Models Used

- Logistic Regression
- Random Forest
- XGBoost
- KNN
- SVC
- Linear Regression
- Decision Tree Regressor
- SVR
